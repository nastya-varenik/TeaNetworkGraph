Тема: Графовая база данных для сети любителей чая
Цель: Исследование связей между сортами чая, странами происхождения, производителями, ароматами, типами заваривания и предпочтениями употребления.  

Создайте графовую базу данных в MS SQL Server, которая моделирует социальную сеть любителей чая. База данных должна хранить информацию о людях, странах и видах чая, а также связях между ними, таких как дружба, место жительства, производство чая и предпочтения в чае. Используйте графовые возможности MS SQL Server для создания узлов и рёбер, заполните таблицы данными, напишите запросы для анализа сети, создайте диаграммы и визуализируйте граф в Power BI.

Требования:  
- Создание графовой БД в MS SQL Server  
- Не менее 3 таблиц узлов и 3 таблиц рёбер  
- Заполнение узлов минимум 10 строками  
- Установление связей через рёбра  
- Реализация запросов MATCH и SHORTEST_PATH  
- Построение диаграмм в draw.io  
- Визуализация в Power BI (Force-Directed Graph)
